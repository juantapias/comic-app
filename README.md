# Comic App
To start with this project you need follow this steps
* Clone repo
```
git clone git@github.com:juantapias/comic-app.git
```
* Folder app
```
cd comic-app
```
* install all npm's dependencies
```
npm install
```
* Start with npm
```
npm start
```
## Note
